# Practicum Coffee Shop

This is the landing page for the Practicum Coffee Shop, a cozy spot where programming meets coffee culture.

## Project Features

* Semantic HTML5
* Flexbox and Grid layout
* Positioning
* Flat BEM file structure
* A custom form
* CSS animation and transform
* Responsive design for different screen sizes:
  * 1440px and higher (standard laptops and beyond)
  * From 768px to 1439px (from a tablet to a laptop)
  * From 320px to 767px (from a mobile device to a tablet)

## Project Functionality

The webpage features:
* Interactive navigation menu
* Responsive menu section with coffee drinks and treats
* Custom recipe videos embedded from YouTube
* A reservation form for booking tables
* About section with animated elements
* Social media links in the footer

## Technologies and Techniques Used

* HTML5
* CSS3
* BEM Methodology
* Flexbox
* Grid Layout
* Positioning
* Media Queries for responsiveness
* CSS Animations
* Form validation
* Embedded content
